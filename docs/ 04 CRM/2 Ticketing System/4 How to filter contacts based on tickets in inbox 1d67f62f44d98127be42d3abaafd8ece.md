# How to filter contacts based on tickets in inbox

[[https://drive.google.com/file/d/1fc2GUmAuU1EpBYuglOdIBShWrNNJnMhN/view?usp=drive_link](https://drive.google.com/file/d/1fc2GUmAuU1EpBYuglOdIBShWrNNJnMhN/view?usp=drive_link)](https://drive.google.com/file/d/1fc2GUmAuU1EpBYuglOdIBShWrNNJnMhN/view?usp=drive_link)

To filter tickets in the inbox, follow these steps:

1. Navigate to the Filter Section
    - Go to the filter section in the inbox.
2. Filter by Specific Board Stage
    - Select the specific board stage you want to filter.
