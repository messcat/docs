# How to create a ticket?

You have three options for creating a ticket:

### Option 1: Create a Ticket from the CRM Page

[[https://drive.google.com/file/d/12WYGwlyO3wT5LgTxbDaNf5dnCDqDl99M/view?usp=drive_link](https://drive.google.com/file/d/12WYGwlyO3wT5LgTxbDaNf5dnCDqDl99M/view?usp=drive_link)](https://drive.google.com/file/d/12WYGwlyO3wT5LgTxbDaNf5dnCDqDl99M/view?usp=drive_link)

1. Navigate to the CRM page.
2. Locate the plus icon above each board stage.
3. Click on the plus icon to create a ticket in that stage.
4. You will be prompted to select a contact and write the ticket title.

### Option 2: Create a Ticket from the Inbox Page

[[https://drive.google.com/file/d/1Je_2if_o7-GCjRVeFaY3xnjm0o4fZ-ww/view?usp=drive_link](https://drive.google.com/file/d/1Je_2if_o7-GCjRVeFaY3xnjm0o4fZ-ww/view?usp=drive_link)](https://drive.google.com/file/d/1Je_2if_o7-GCjRVeFaY3xnjm0o4fZ-ww/view?usp=drive_link)

1. Navigate to the inbox page.
2. Locate the create ticket icon on the top right.
3. Click on the create ticket icon.
4. You will be prompted to select the board and stage, as well as write the ticket title.

### Option 3: Create a Ticket Using Triggers and App Nodes in Message Flows

1. Use triggers and app nodes within your message flows to automate the creation of new tickets.
2. For detailed instructions, refer to the [documentation on using triggers and app nodes](https://help.messcat.ai/message-flows/kSYFmwLcqbtLQQAhJb4Sek/how-to-use-the-create-ticket-node/gs1JR1RbELeWv8zzPh3vvF).
