# How to create/update tickets automatically through message flows


## Automating Ticket Management Using Triggers and App Nodes

To create, update, and manage tickets automatically without manual intervention, you can utilise triggers and app nodes in the message flow module.

### App Nodes for Ticket Management

- **Create Ticket App Node**: Allows you to automatically create new tickets. [Learn more](https://help.Messcat.tech/message-flows/kSYFmwLcqbtLQQAhJb4Sek/how-to-use-the-create-ticket-node/gs1JR1RbELeWv8zzPh3vvF)
- **Update Ticket App Node**: Enables you to update existing ticket details. [Learn more](https://help.Messcat.tech/message-flows/kSYFmwLcqbtLQQAhJb4Sek/how-to-use-the-update-ticket-node/wdSHptD6CreTaneMXs78Hc)
- **Stop Ticket Timer App Node**: Manages ticket timers automatically. [Learn more](https://help.Messcat.tech/message-flows/kSYFmwLcqbtLQQAhJb4Sek/how-to-use-the-stop-ticket-timer-node/iDWDGVu9mhHGvb2Fvq9JUz)

### Triggers for Ticket Management

- **Ticket Created Trigger**: Activates when a new ticket is created. [Learn more](https://help.Messcat.tech/message-flows/kSYFmwLcqbtLQQAhJb4Sek/ticket-updated-trigger/qssdZ3yFGcH8fWzjwffNr7)
- **Ticket Updated Trigger**: Triggers upon updating an existing ticket. [Learn more](https://help.Messcat.tech/message-flows/kSYFmwLcqbtLQQAhJb4Sek/new-ticket-created-trigger/wzSCYPzRPq3BkEtpfnPz5a)

This feature allows for seamless automation of ticket creation, updates, and management within your system.
