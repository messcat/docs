# How to Tag a Contact by Message History?

The **Tag by Message History** feature allows you to automatically assign tags to contacts based on specific keywords found in your chat history. This helps you categorize contacts efficiently and target them more effectively.

---

## Steps

### **1. Access the Tagging Feature**

Follow these steps to access the **Tag by Message History** feature:

1. Open the **Contacts** section within your app.
2. Tap the three-dot menu in the top right corner.
3. From the menu, select **Tag by Message History**.
    
    ![](How%20to%20Tag%20a%20Contact%20by%20Message%20History/tag.png)
    

### **2. Set Up Your Tagging Criteria**

Now, define the criteria for tagging:

1. In the **Message Text** field, enter the keywords (e.g., “Insurance”) that should trigger the tag assignment.
2. Select a relevant **tag** from the existing list or create a **new tag** to categorize the contacts.
3. Use the **Include** option to filter the messages:
    - **All** messages, if direction doesn’t matter.
    - **Only sent by me**, to tag messages you sent.
    - **Only received by me**, to tag messages you received.
4. Optionally, set a **date range** (e.g., past month) to focus on recent interactions.
    
    ![](How%20to%20Tag%20a%20Contact%20by%20Message%20History/tag_2.png)
    

### **3. Apply the Tags**

After setting your criteria, click the **Tag** button to begin the tagging process.

### **4. Tagging Completion**

Once the tagging process is complete, you’ll see a success message confirming that the tags have been applied.

<aside>
❗ **Please note:** If you're tagging a large number of contacts, there might be a wait time of up to 2 minutes.

</aside>
