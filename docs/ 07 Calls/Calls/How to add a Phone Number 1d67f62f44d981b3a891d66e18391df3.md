# How to add a Phone Number

### Overview

The Phone Numbers page allows you to manage the phone numbers associated with your account for calling purposes. You can add, verify, and manage phone numbers through this interface.

[[https://drive.google.com/file/d/1iz4xnkaLekuHjDDUuzf8VufXBzcHnjPA/view?usp=sharing](https://drive.google.com/file/d/1iz4xnkaLekuHjDDUuzf8VufXBzcHnjPA/view?usp=sharing)](https://drive.google.com/file/d/1iz4xnkaLekuHjDDUuzf8VufXBzcHnjPA/view?usp=sharing)

### Adding a Phone Number

To add a new phone number, follow these steps:

1. **Click "Add your phone number"**:
    - Navigate to the Phone Numbers page and click on the button labeled "Add your phone number".
2. **Enter Details in the Dialog Box**:
    - A dialog box will appear prompting you to enter your phone number and name.
    - Fill in the fields with the relevant information.
    - Click on "Get Code" to generate a verification code.
3. **Verify Your Phone Number**:
    - A code will appear in the dialog box.
    - When you receive the verification call, enter this code using the number pad on your phone.
    - Click "Confirm" to complete the verification process.

### Managing Phone Numbers

- **Edit Name**:
    - You can edit the name associated with the phone number by clicking the "Edit" button.
- **Delete Phone Number**:
    - If you need to change the phone number, you must delete the existing number and add a new one.
    - To delete, click the "Delete" button next to the phone number.
