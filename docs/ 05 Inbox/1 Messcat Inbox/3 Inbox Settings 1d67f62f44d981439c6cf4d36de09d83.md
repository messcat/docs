# Inbox Settings

There are some settings which can tailor your experience of using the Shared Team Inbox. Read this article to find out the different settings that exist and what they do. 

![download.png](Inbox%20Settings/download.png)

## Features

### Incognito Mode

This feature allows you to **view messages** in chats without changing their status to '**read**'. This is useful for team admins to go over chats without accidentally marking them as read. 

![incognito mode.gif](Inbox%20Settings/incognito_mode.gif)

### Show/Hide Channel Name

You can use this setting to switch between displaying your channel's name or phone number in the inbox.

![channel gif.gif](Inbox%20Settings/channel_gif.gif)

### Show/Hide Full Date

You can use this setting to change the format of the date when messages were sent or recieved.

![Untitled design (1).gif](Inbox%20Settings/Untitled_design_(1).gif)

### Hide/Show Scheduled Messages

You can use this setting to hide/show any messages that are pending to be sent.

![Inbox Setting- schedule message.gif](Inbox%20Settings/Inbox_Setting-_schedule_message.gif)

### Auto-Suggest Replies

This setting enables you to easily switch the automatic AI-suggested replies on or off.

![AI replies.gif](Inbox%20Settings/AI_replies.gif)

### Warn on Editing Message Flows

This setting provides an alert when there are modifications made to the message flow.

![warning flow.gif](Inbox%20Settings/warning_flow.gif)
