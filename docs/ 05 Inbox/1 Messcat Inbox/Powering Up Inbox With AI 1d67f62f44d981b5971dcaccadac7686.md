# Powering Up Inbox With AI


## Transcribe Voice Messages

AI Transcription allows automatic transcription of voice messages for easier automation and response handling.

### **Manual Transcription**

1. Locate a voice message
2. Hover over the message
3. Click the "Transcribe" icon button to generate the transcript

### **Auto Transcription & Automation**

Using auto transcription, you can setup automation with voice messages as well. Auto transcription will automatically transcribe all incoming voice messages. This transcribed text can be used in all automations like keyword reply & triggers — without any changes required

**How to enable auto transcription?**

1. Auto transcription is a channel-wide setting, navigate to the channels page
2. Find the channel you want to enable auto transcription for, open “advanced settings”
3. Enable "Auto Transcribe Voice Messages"
    
    ![Screenshot 2025-02-20 at 1.32.29 PM.png](Powering%20Up%20Inbox%20With%20AI/Screenshot_2025-02-20_at_1.32.29_PM.png)
    

## Translate Messages

1. Locate the message you would like to translate
2. Hover over the message to find the “translate” button
    
    ![Screenshot 2025-02-20 at 2.04.31 PM.png](Powering%20Up%20Inbox%20With%20AI/Screenshot_2025-02-20_at_2.04.31_PM.png)
    
3. Click the translate button, which will translate to your default language
    
    ![Screenshot 2025-02-20 at 2.04.50 PM.png](Powering%20Up%20Inbox%20With%20AI/Screenshot_2025-02-20_at_2.04.50_PM.png)
    
4. If you’d like to translate to another language, you can click the button again & select another language

## FAQ

### 1. What languages are supported for transcription?

Chinese, English, Malay, Portuguese, Spanish, Hindi, are all supported. A voice note can also contain multiple languages, though this may result in reduced accuracy of the transcription

### 2. How to use transcribed messages in triggers?

Transcribed messages behave the same way as regular text messages, so you can utilise the “text” property to run any automation via triggers
