# How to Schedule a message in Inbox?

To schedule a message, head over to the inbox, type a message and then click on the small downward arrow on the send button, this will open a dropdown to let you choose a desired time. 

![output.gif](How%20to%20Schedule%20a%20message%20in%20Inbox/output.gif)

---

### Cancel if reply received

When using a scheduled message to follow up on a client, in some instances the client may respond before the scheduled message is sent, in that case the scheduled message will be cancelled automatically.
