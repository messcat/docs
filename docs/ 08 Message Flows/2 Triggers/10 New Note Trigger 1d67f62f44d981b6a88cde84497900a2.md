## Overview

![Screenshot 2025-03-11 at 1.01.58 PM.png](New%20Note%20Trigger/Screenshot_2025-03-11_at_1.01.58_PM.png)

The **New Note** trigger allows you to automate workflows whenever a new note is added to a conversation. This trigger is useful for tracking important updates, notifying team members, or integrating notes with external systems.

To learn more about how to add and use notes, click on [this](https://help.Messcat.tech/inbox-/fZtwDNaJhZrhvo3UfHtK5f/how-to-use-notes/pBi35hZkk715xDuUZTomkx) link

## Use Cases

- **Internal Notifications**: Alert team members when a new note is added to a high-priority conversation.
- **CRM Updates**: Automatically update CRM records when a new note is logged with certain keywords.
- **Ticket Escalation**: Trigger an escalation workflow if a note contains specific keywords.
- **External Syncing**: Sync notes with third-party tools like Google Sheets, Slack, or a project management app.

## Trigger Conditions

You can customize the trigger with the following conditions:

- **Note Content**: Contains specific words or phrases.
- **Author**: Notes added by specific users or roles.
- **Conversation/Ticket Status**: Trigger only if the associated item is open, pending, or closed.
- **Tagged Notes**: Only activate for notes with a specific tag.

## Example workflow

### **Scenario: Notify a Manager When a Critical Note is Added**

1. **Trigger**: New Note is added.
2. **Condition**: If the note contains "urgent" or "escalate".
3. **Action**: Send a Slack notification to the manager.
4. **Action**: Send an in-notification to teammate on Messcat
