# How to use the Create Ticket node



The “Create Ticket” node can be used to automatically create new tickets in any CRM board. This can help in streamlining customer service and support processes by ensuring that inquiries or issues are promptly tracked and managed

- **Customer Support Inquiries**: Automatically create a ticket when a customer sends a message indicating they need support, ensuring that no inquiry goes unattended.
- **Sales Leads**: Generate a ticket for new sales leads captured through a form or chat, allowing your sales team to follow up promptly.
- **Feedback and Reviews**: Create tickets for customer feedback or reviews, making it easier to track and respond to customer opinions and suggestions.

### How to use the Create Ticket node

[[https://drive.google.com/file/d/1JVEt048ZwkcmyaqF5IaeOOJUQiCyOOTt/view?usp=drive_link](https://drive.google.com/file/d/1JVEt048ZwkcmyaqF5IaeOOJUQiCyOOTt/view?usp=drive_link)](https://drive.google.com/file/d/1JVEt048ZwkcmyaqF5IaeOOJUQiCyOOTt/view?usp=drive_link)

Step 1: Setup your message flow and navigate to the node where you want to create a new ticket

Step 2: Click on “Choose Next Step” and select “Create Ticket”

<aside>
❗ Create Ticket node can be used in after buttons, conditions or triggers

</aside>

Step 3: Enter the title of your ticket. You can manually input a name or use any custom variable to personalise the ticket

Step 4: Select the CRM board and stage where you want to create your new ticket

<aside>
❗ The CRM board and stages should already be created before this step. Please click [here](https://help.Messcat.tech/crm/8L5CutnFfW9kjkXP3WPyKa/how-to-create-and-manage-crm-boards/rhEGYCLdWoawMqx6D9t2e1) to learn how

</aside>

Step 5(Optional): Set a timer for the ticket. This indicates how much time your teammate has to respond to/close the ticket. Leave empty if you don’t want to set a timer
