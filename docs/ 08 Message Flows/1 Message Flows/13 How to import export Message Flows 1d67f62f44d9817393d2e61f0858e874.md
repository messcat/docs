# How to import/export Message Flows

Article Description: Learn how to import or export existing message flows them for backup or reuse


Message Flows allow you to automate conversations based on specific triggers and conditions. To streamline workflow management, you can import existing message flows or export them for backup or reuse.

---

## Exporting a Message Flow

Step 1: Hover over the navigation bar on the left side to find the “Automation” section. Then, navigate to Message Flows

![Screenshot 2025-02-24 at 4.41.57 PM.png](How%20to%20import%20export%20Message%20Flows/Screenshot_2025-02-24_at_4.41.57_PM.png)

Step 2: Locate the message flow(s) you want to export and select them using the checkbox on the left side

![Screenshot 2025-02-24 at 4.45.13 PM.png](How%20to%20import%20export%20Message%20Flows/Screenshot_2025-02-24_at_4.45.13_PM.png)

Step 3: Click on the “Export” button below the search bar

![Screenshot 2025-02-24 at 4.46.31 PM.png](How%20to%20import%20export%20Message%20Flows/Screenshot_2025-02-24_at_4.46.31_PM.png)

Step 4: Select whether you want to include the values of the flow variables with your export or not. Find out more about flow variables [here](https://help.messcat.ai/message-flows/kSYFmwLcqbtLQQAhJb4Sek/flow-variables/vPrCvh6mu2cFo7q2iwP2aQ). Then, click “Export”

![Screenshot 2025-02-24 at 4.48.37 PM.png](How%20to%20import%20export%20Message%20Flows/Screenshot_2025-02-24_at_4.48.37_PM.png)

You will now have a “.yaml” file downloaded. You can use this as a backup of your flows or to import them into a different team

---

## Importing a Message Flow

Step 1: Hover over the navigation bar on the left side to find the “Automation” section. Then, navigate to Message Flows

![Screenshot 2025-02-24 at 4.41.57 PM.png](How%20to%20import%20export%20Message%20Flows/Screenshot_2025-02-24_at_4.41.57_PM.png)

Step 2: Click on the “Import” button on the top right

![Screenshot 2025-02-24 at 4.50.18 PM.png](How%20to%20import%20export%20Message%20Flows/Screenshot_2025-02-24_at_4.50.18_PM.png)

Step 3: Upload your “.yaml” file of the flows you had previously exported. 

![Screenshot 2025-02-24 at 4.51.53 PM.png](How%20to%20import%20export%20Message%20Flows/Screenshot_2025-02-24_at_4.51.53_PM.png)

Step 4: Select which flows you want to upload from the file and then click “Import”. 

- You can also preview the flows before uploading them
- Select “Include” to import all the selected flows
- Select “Exclude” to import all the flows other than the selected ones

![Screenshot 2025-02-24 at 4.53.21 PM.png](How%20to%20import%20export%20Message%20Flows/Screenshot_2025-02-24_at_4.53.21_PM.png)
