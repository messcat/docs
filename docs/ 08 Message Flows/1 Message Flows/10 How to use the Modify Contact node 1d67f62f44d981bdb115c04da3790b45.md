# How to use the Modify Contact node



The "Modify Contact" node allows you to automate the process of updating contact information. This includes adding or removing tags, setting custom fields, and assigning chats to teammates or using round-robin assignment between multiple teammates.

![image.png](How%20to%20use%20the%20Modify%20Contact%20node/image.png)

- **Lead Qualification:** Automatically add a "Qualified Lead" tag and set the custom field "Lead Score" to "High" when a contact meets certain criteria.
- **Customer Segmentation:** Remove the "New Customer" tag and add the "Returning Customer" tag when a customer makes a repeat purchase.
- **Team Assignment:** Assign chats from high-priority customers to senior support agents and distribute other chats evenly among the support team using round-robin.

<aside>
❗ Modify Contact node can be used in after buttons, conditions or triggers

</aside>

### How to use the Modify Contact node

- **Name -** This can be used to change the name of your contact. You can manually input a name or use a custom field
- **Add tag -** Select the tag that you want to add from the dropdown
- **Remove tag -** Select the tag that you want to remove from the dropdown
- **Set custom field -** Set or update custom fields for the contact by clicking “Add item”, selecting your custom field from the dropdown, and then setting its value
- **Remove custom field -** Clear out the custom field for the contact by selecting it from the dropdown
- **Assignee -** Select a specific teammate to assign the chats to. Select multiple teammates evenly distribute chats amongst them using round robin method
- **Remove Assignee -** Enable this option to remove the assignee from the contact
