# Default Dashboard Overview Latest

This document provides an overview of the dashboard metrics, breaking down the components and their meanings. Use this as a guide to understand and interpret the dashboard data.

---

## 1. Chat Metrics

![image.png](Default%20Dashboard%20Overview%20Latest/image.png)

### Total Chats

- **Definition**: Displays the total number of chats initiated within the selected timeframe.
- **Example**: 978 total chats.

### New Chats

- **Definition**: Shows the number of new chats started.
- **Example**: 13 new chats.

### Chat Growth Rate

- **Definition**: The percentage increase or decrease in the number of chats over time.
- **Example**: 1% growth.

### Active Chats Today

- **Definition**: The number of chats that are actively ongoing for the day.
- **Example**: 6 active chats.

### Contact Velocity

- **Definition**: The percentage increase or decrease in the number of contacts over time.
- **Example**: 1% velocity.

---

## 2. Contact Metrics

![image.png](Default%20Dashboard%20Overview%20Latest/image%201.png)

### Total Contacts

- **Definition**: Total number of contacts recorded in the system.
- **Example**: 11,273 contacts.

### Contacts Active Today

- **Definition**: Number of contacts engaging with chats today.
- **Example**: 6 active contacts.

### Chat Activity Rate

- **Definition**: The percentage of contacts actively participating in chats.
- **Example**: 4% activity rate.

### Incoming Messages

- **Definition**: Total number of incoming messages received.
- **Example**: 389 incoming messages.

### New Contacts By Channel

- **Definition**: Breaks down new contacts by their communication channel.
    - **Example**:
        - WhatsApp: 7
        - Messenger: 3
        - Instagram: 1
        - Email: 1
        - SMS: 1

## 3. Message Metrics

![image.png](Default%20Dashboard%20Overview%20Latest/image%202.png)

### Total Messages

- **Definition**: Total number of messages exchanged.
- **Example**: 28,773 messages.

### Incoming Messages to Outgoing Messages Ratio

- **Definition**: Compares incoming to outgoing message volumes.
- **Example**: Ratio of 1.597.

### Contact Activity Level

- **Definition**: Percentage of contacts engaging with messages.
- **Example**: 0% activity level.

### Outgoing Messages

- **Definition**: Total number of outgoing messages sent.
- **Example**: 625 outgoing messages.

### Messages Sent by Team Members

- **Definition**: Tracks how many messages were sent by specific team members or automation.
    - **Example**:
        - Automation: 267
        - Aditya Gupta: 247
        - CD - Harvard Chin: 44
        - Unknown: 15
        - Sameera N: 10

### Average Incoming Messages Per Contact

- **Definition**: Shows the average number of incoming messages per contact.
- **Example**: 11.718 messages.

### Average Outgoing Messages Per Contact

- **Definition**: Displays the average number of outgoing messages per contact.
- **Example**: 18.692 messages.

### Average Response Time

- **Definition**: Average time it takes to respond to an incoming message.
- **Example**: 18 minutes.

### First Assign Time

- **Definition**: Time taken to assign the first message to a team member.
- **Example**: 0 seconds.

## **CRM Metrics**

![image.png](Default%20Dashboard%20Overview%20Latest/image%205.png)

### **Average Tags per Contact:**

The average number of tags assigned per contact to categorize or organize them.

### **In and Out Ratio:**

Ratio of inbound to outbound communication.

### **Tags Breakdown:**

A visual representation (pie chart) of tag distribution, highlighting the most frequently used tags like “WhatsApp Broadcast,” “CD Broadcast,” and custom tags.

## Notes

- Metrics update based on the chosen time filter (e.g., daily, weekly, monthly).
