# Default Dashboard Overview (OLD)

**Default Dashboard Overview**

![image.png](Default%20Dashboard%20Overview%20(OLD)/image.png)

When you create your first dashboard, you'll see a set of default widgets that give you insights into key metrics for your team’s performance. Here’s what each graph shows:

1. **New Chats Breakdown**
    
    This chart displays the number of new chats initiated within the selected time period. It breaks down chats into two categories:
    
    - **Incoming**: Chats started by your customers.
    - **Outgoing**: Chats initiated by your team.The percentage change from the previous cycle is also displayed to give a quick overview of the trend in new chats.
    
    ![image.png](Default%20Dashboard%20Overview%20(OLD)/image%201.png)
    
2. **Messages Exchanged Breakdown**
    
    This chart shows the total number of messages exchanged on the platform during the selected time period. Like the New Chats Breakdown, it is split into:
    
    - **Incoming**: Messages received by your team.
    - **Outgoing**: Messages sent by your team.The percentage change from the last cycle helps you track whether message activity is increasing or decreasing.
    
    ![image.png](Default%20Dashboard%20Overview%20(OLD)/image%202.png)
    
3. **Message Open Rate**
    
    This section shows the percentage of messages that were opened (or read) by recipients. You’ll also see a breakdown by different categories like:
    
    - **WA Business**: Messages sent via WhatsApp Business.
    - **WABA Malaysia**: WhatsApp Business API in Malaysia.
    - Other channels as configured.The red arrow indicates the percentage decrease compared to the previous cycle.
    
    ![image.png](Default%20Dashboard%20Overview%20(OLD)/image%203.png)
    
4. **Messages Sent via Automation**
    
    This chart tracks the total number of messages sent through automated flows or notifications. It provides a breakdown into two categories:
    
    - **Message Flow**: Automated messages triggered by workflows.
    - **Notifications**: Pre-set messages such as reminders or alerts.The percentage change helps you assess the effectiveness and usage of your automation.
    
    ![image.png](Default%20Dashboard%20Overview%20(OLD)/image%204.png)
    
5. **Messages Sent by Team Members**
    
    This graph displays the total number of messages sent by different team members, allowing you to compare individual performance. You’ll see a breakdown of:
    
    - **Automation**: Messages sent via automation.
    - **Admin**: Messages sent by administrators.
    - **Other team members** by name.The percentage change from the last cycle highlights shifts in team member activity.
    - None indicates messages sent outside Messcat.
    
    ![image.png](Default%20Dashboard%20Overview%20(OLD)/image%205.png)
    
6. **Average Reply Speed**
    
    This chart shows the average time it takes your team to reply to incoming messages. You’ll also see the average reply time for each team member or automated system:
    
    - **None**: Represents messages sent outside Messcat.
    - **Automation**: Average reply speed for automated replies.
    - Individual team members by name. A red arrow indicates that the average reply speed has increased compared to the previous cycle, while a green arrow would show an improvement.
    
    ![image.png](Default%20Dashboard%20Overview%20(OLD)/image%206.png)
    

This default dashboard provides a comprehensive view of your team's messaging performance, helping you quickly understand where improvements or adjustments might be needed.
