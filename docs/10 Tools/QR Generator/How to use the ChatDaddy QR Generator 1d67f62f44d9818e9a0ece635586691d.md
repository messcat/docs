# How to use the Messcat QR Generator?

Last Updated: January 3, 2024


[https://www.loom.com/share/bfa9048a664d48809ccf2640d42cee21?sid=3d8d4fbb-b58f-4de4-91f1-977f158133a3](https://www.loom.com/share/bfa9048a664d48809ccf2640d42cee21?sid=3d8d4fbb-b58f-4de4-91f1-977f158133a3)

Steps

1. Open Messcat dashboard.
2. Locate "Tools" in the left menu.
3. Click **QR Generator**.
4. Dimensions
    - Select a pre-defined size or customize with size.
        
        ![2024-05-07 11_19_02-Window.png](How%20to%20use%20the%20ChatDaddy%20QR%20Generator/481961d7-a661-44a2-987f-deb4ef5bdb53.png)
        
5. **Channel**
    - Select the channel you want customers to message to.
6. **Pre-filled Message**
    - Craft a message that will automatically appear when a user scans the QR code and opens the WhatsApp chat. This helps initiate the conversation and embed specific keywords you might want.
7. **Change Background Photo (Optional)**
8. **Caption (Optional)**
    - Add a call to action text displayed below the QR code.
9. **Caption Text Color (Optional, if caption used)**
10. Click **Download** to save the finalized QR code image.
11. Or you can copy the link generated.
